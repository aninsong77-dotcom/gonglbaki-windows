변환이 안 될 때
먼저 확인 변환 버튼을 눌러도 아무 반응이 없어요 ▼
증상: 버튼을 눌렀는데 로딩도 없고 텍스트도 안 나옴
  • 설정(⚙) → 변환 엔진을 확인해주세요. faster-whisper 또는 whisper.cpp 중 하나여야 해요.
  • 선택한 엔진의 모델이 설치되어 있는지 확인하세요. 설치 안 된 모델로는 시작이 안 돼요.
  • whisper.cpp 엔진이 선택되어 있다면 → faster-whisper로 바꿔서 다시 시도해보세요.
  • 그래도 안 되면 앱을 완전히 종료 후 다시 실행해주세요.
💡 설정을 바꾼 적 없는데 안 된다면 앱 재실행을 먼저 해보세요.
먼저 확인 변환하다가 중간에 멈춰요 ▼
증상: 텍스트가 나오다가 멈추고 더 이상 진행 안 됨
  • 화면에 '이어서 변환' 버튼이 나타나면 눌러주세요. 멈춘 지점부터 이어서 변환할 수 있어요.
  • 파일이 1시간 이상이면 30분 이하로 잘라서 순서대로 변환해보세요.
  • RAM이 8GB 이하라면 설정에서 Small 모델로 바꿔서 시도해보세요.
  • 크롬, 한글 등 다른 프로그램을 모두 닫고 다시 시도해보세요.
확인 필요 앱이 실행은 되는데 흰 화면만 나와요 ▼
증상: 앱 아이콘 클릭 후 창은 뜨는데 흰 화면이거나 멈춤
  • 잠시 기다려보세요. 첫 실행 시 내부 서버 준비에 10~30초 걸릴 수 있어요.
  • 30초 이상 흰 화면이면 앱을 완전히 종료하고 다시 실행하세요.
  • 그래도 같으면 작업 관리자(Ctrl+Shift+Esc)에서 곤글박이 관련 프로세스를 모두 종료 후 다시 실행하세요.
모델 · 다운로드 문제
확인 필요 처음 실행할 때 AI 모델 다운로드가 안 돼요 ▼
증상: 첫 실행 시 다운로드 화면에서 멈추거나 오류
  • 인터넷 연결을 확인해주세요. 첫 실행 시 약 1.5GB 다운로드가 필요해요.
  • VPN이 켜져 있다면 VPN을 끄고 다시 시도해보세요.
  • C드라이브에 3GB 이상 여유 공간이 있는지 확인해주세요.
  • 다운로드가 중간에 끊겼다면 앱을 재실행하면 이어서 다운로드돼요.
💡 다운로드가 완료되면 이후엔 인터넷 없이 사용 가능해요.
확인 필요 변환 속도가 너무 느려요 ▼
증상: 변환은 되는데 실시간의 3배 이상 시간이 걸림
  • 설정에서 모델을 Small로 바꿔보세요. Medium보다 2~3배 빨라요.
  • 변환 중 다른 무거운 프로그램은 모두 종료해주세요.
  • 파일을 15~20분 단위로 나눠서 변환하면 더 안정적이에요.
💡 NVIDIA GPU가 있는 PC는 자동으로 GPU가 사용돼요. GPU 드라이버가 최신인지 확인해보세요.
회사 · 학교 지급 PC / 보안 프로그램이 강한 환경
높은 가능성 회사/학교 PC에서만 변환이 안 돼요 ▼
증상: 개인 PC에선 잘 되는데 기관 PC에서만 안 됨
  • 백신·보안 프로그램이 내부 실행파일을 차단하고 있을 수 있어요. 백신을 열어 차단된 항목 또는 격리된 파일을 확인해보세요.
  • 백신에서 곤글박이 설치 폴더 전체를 검사 예외(화이트리스트)에 추가해보세요.
  • 설정(⚙) → 엔진을 faster-whisper로 바꿔보세요. whisper.cpp보다 차단될 가능성이 낮아요.
  • 그래도 안 되면 IT 담당자에게 실행 권한을 요청하거나 개인 노트북 사용을 권장드려요.
⚠️ 기관 PC는 외부 실행파일 실행 자체가 정책으로 막혀 있는 경우가 있어요.
높은 가능성 whisper.cpp 엔진으로 변환하면 아무 반응이 없어요 ▼
증상: whisper.cpp 엔진 선택 후 변환 시도 시 무반응
  • 파일 탐색기에서 C:\whisper-models 폴더가 있는지 확인해보세요.
  • 없다면 직접 만들어보세요. 권한 오류가 뜨면 C드라이브 쓰기 권한이 없는 것이에요.
  • 이 경우 설정에서 faster-whisper 엔진으로 전환하면 해결돼요. faster-whisper는 이 경로를 사용하지 않아요.
💡 faster-whisper 엔진은 C드라이브 루트 폴더를 사용하지 않아요.
높은 가능성 설치 시 파란 경고창이 뜨고 막혀요 ▼
증상: "Windows가 PC를 보호했습니다" 또는 "알 수 없는 게시자" 경고
  • 경고창에서 '추가 정보'를 클릭해주세요.
  • '실행' 버튼이 나타나면 클릭해서 설치를 진행하세요.
  • '실행' 버튼이 아예 없다면 기관 보안 정책으로 외부 소프트웨어 설치 자체가 차단된 거예요. IT 담당자 문의가 필요해요.
⚠️ 곤글박이는 코드 서명 인증서가 없어 SmartScreen 경고가 정상적으로 표시돼요. 악성 소프트웨어가 아니에요.
확인 필요 첫 실행 시 모델 다운로드가 시작도 안 돼요 (기관 네트워크) ▼
증상: 다운로드 화면에서 아무 진행도 없음
  • 기관 네트워크는 외부 대용량 다운로드를 차단하는 경우가 있어요.
  • 스마트폰 핫스팟으로 바꿔서 다운로드해보세요. 약 1.5GB 데이터가 필요해요.
  • 다운로드 완료 후에는 인터넷 없이 사내 네트워크에서도 변환 가능해요.
💡 AI 모델은 최초 1회만 다운로드하면 돼요.
참고 오래된 PC (RAM 8GB 이하, 구형 CPU) ▼
증상: 변환이 매우 느리거나 중간에 앱이 멈춤
  • 설정에서 모델을 Small로 바꿔주세요. 메모리 사용량이 크게 줄고 속도가 빨라져요.
  • 변환 전 크롬, 한글 등 다른 앱을 모두 종료해주세요.
  • 파일을 15~20분 단위로 나눠서 순서대로 변환하는 것을 권장해요.
  • 변환 중 PC를 다른 용도로 사용하지 마세요.
⚠️ 10년 이상 된 노트북에서 1시간 녹음 변환 시 2~3시간이 걸릴 수 있어요. Small 모델 사용 시 30~60분으로 단축 가능해요.
