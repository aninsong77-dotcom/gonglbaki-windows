검색 결과가 없어요. 다른 낱말로 찾아보시거나, 아래 문제 해결 탭을 확인해보세요.
변환이 안 될 때
먼저 확인 변환 버튼을 눌러도 아무 반응이 없어요 ▼
증상: 파일을 선택했는데 로딩도 없고 텍스트도 안 나옴
  • 설정(⚙) → 변환 엔진이 다운로드되어 있는지 확인하세요 (large-v3-turbo, 약 1.6GB).
  • 모델이 없다면 설정 화면의 다운로드 버튼으로 받아주세요. 다운로드가 끝나야 변환이 시작돼요.
  • 그래도 안 되면 앱을 완전히 종료 후 다시 실행해주세요.
💡 설정을 바꾼 적 없는데 안 된다면 앱 재실행을 먼저 해보세요.
먼저 확인 변환하다가 중간에 멈춰요 ▼
증상: 텍스트가 나오다가 멈추고 더 이상 진행 안 됨
  • 화면에 '이어서 변환' 배너가 나타나면 같은 음성 파일을 다시 선택해주세요. 멈춘 지점부터 이어서 변환돼요.
  • 절전 모드로 컴퓨터가 잠들면서 중단됐을 가능성이 높아요. 변환 중에는 절전이 안 되게 설정해두면 좋아요.
  • 크롬, 한글 등 다른 무거운 프로그램을 모두 닫고 다시 시도해보세요.
확인 필요 앱이 실행은 되는데 흰 화면만 나와요 ▼
증상: 앱 아이콘 클릭 후 창은 뜨는데 흰 화면이거나 멈춤
  • 잠시 기다려보세요. 첫 실행 시 내부 서버 준비에 10~30초 걸릴 수 있어요.
  • 30초 이상 흰 화면이면 앱을 완전히 종료하고 다시 실행하세요.
  • 그래도 같으면 작업 관리자(Ctrl+Shift+Esc)에서 곤글박이 관련 프로세스를 모두 종료 후 다시 실행하세요.
변환 결과 문제
참고 같은 문장이 반복돼서 나와요 ▼
증상: 특정 구간부터 같은 말이 계속 반복됨 (긴 침묵 구간 근처에서 자주 발생)
  • 2026년 7월 업데이트로 무음 제거·반복 필터가 적용돼 크게 줄었어요. 최신 설치파일인지 확인해보세요.
  • 그래도 심하면 디버그 로그를 복사해 문의해주세요 (음성·변환 내용은 로그에 포함되지 않아요).
모델 · 다운로드 문제
확인 필요 처음 실행할 때 AI 모델 다운로드가 안 돼요 ▼
증상: 다운로드 화면에서 멈추거나 오류
  • 인터넷 연결을 확인해주세요. 변환 모델만 약 1.6GB, 요약 모델까지 받으면 추가 2.4GB가 필요해요.
  • VPN이 켜져 있다면 VPN을 끄고 다시 시도해보세요.
  • C드라이브에 5GB 이상 여유 공간이 있는지 확인해주세요.
  • 다운로드가 중간에 끊겼다면 앱을 재실행하면 이어서 다운로드돼요.
💡 다운로드가 완료되면 이후엔 인터넷 없이 사용 가능해요.
확인 필요 변환 속도가 너무 느려요 ▼
증상: 변환은 되는데 실시간의 3배 이상 시간이 걸림
  • 변환 중 다른 무거운 프로그램은 모두 종료해주세요.
  • CPU 코어가 2개 이하인 저사양 PC라면 원래 느릴 수 있어요 — 변환 시작 시 안내 메시지를 확인해주세요.
💡 일반적인 데스크탑 CPU 기준 60분 녹음이 약 30분에 변환됩니다.
분석 · 사례서랍 문제
참고 분석 버튼이 눌러지지 않아요 ▼
증상: 📊 분석 버튼이 비활성화 상태
  • 화자 분리를 먼저 완료해야 분석 버튼이 활성화돼요.
  • 요약까지 쓰려면 설정에서 요약 모델(2.4GB)을 받아야 해요. 안 받아도 빈출 단어는 사용 가능해요.
참고 사례서랍 비밀번호를 잊었어요 ▼
증상: 잠금 화면에서 비밀번호가 기억나지 않음
  • 잠금 화면의 "비밀번호를 잊으셨나요?" → 재설정을 눌러주세요.
  • 이 비밀번호는 파일을 암호화하는 게 아니라 여는 관문이라, 재설정해도 데이터 손실은 없어요.
회사 · 학교 지급 PC / 보안 프로그램이 강한 환경
높은 가능성 회사/학교 PC에서만 변환이 안 돼요 ▼
증상: 개인 PC에선 잘 되는데 기관 PC에서만 안 됨
  • 백신·보안 프로그램이 내부 실행파일을 차단하고 있을 수 있어요. 백신을 열어 차단된 항목 또는 격리된 파일을 확인해보세요.
  • 백신에서 곤글박이 설치 폴더 전체를 검사 예외(화이트리스트)에 추가해보세요.
  • 그래도 안 되면 IT 담당자에게 실행 권한을 요청하거나 개인 노트북 사용을 권장드려요.
⚠️ 기관 PC는 외부 실행파일 실행 자체가 정책으로 막혀 있는 경우가 있어요.
높은 가능성 설치 시 파란 경고창이 뜨고 막혀요 ▼
증상: "Windows가 PC를 보호했습니다" 또는 "알 수 없는 게시자" 경고
  • 경고창에서 '추가 정보'를 클릭해주세요.
  • '실행' 버튼이 나타나면 클릭해서 설치를 진행하세요.
  • '실행' 버튼이 아예 없다면 기관 보안 정책으로 외부 소프트웨어 설치 자체가 차단된 거예요. IT 담당자 문의가 필요해요.
⚠️ 곤글박이는 코드 서명 인증서가 없어 SmartScreen 경고가 정상적으로 표시돼요. 악성 소프트웨어가 아니에요.
확인 필요 첫 실행 시 모델 다운로드가 시작도 안 돼요 (기관 네트워크) ▼
증상: 다운로드 화면에서 아무 진행도 없음
  • 기관 네트워크는 외부 대용량 다운로드를 차단하는 경우가 있어요.
  • 스마트폰 핫스팟으로 바꿔서 다운로드해보세요. 변환 모델만 약 1.6GB예요.
  • 다운로드 완료 후에는 인터넷 없이 사내 네트워크에서도 변환 가능해요.
💡 AI 모델은 최초 1회만 다운로드하면 돼요.
참고 오래된 PC (RAM 8GB 이하, 구형 CPU) ▼
증상: 변환이 매우 느리거나 중간에 앱이 멈춤
  • 변환 전 크롬, 한글 등 다른 앱을 모두 종료해주세요.
  • 변환 중 PC를 다른 용도로 사용하지 마세요.
  • CPU 코어가 2개 이하면 변환 시작 시 저사양 안내가 표시돼요 — 시간이 걸려도 정상 작동이니 기다려주세요.

문제가 해결되지 않을 때 — 디버그 파일

곤글박이는 오류 발생 시 자동으로 디버그 파일을 만들어요. 이 파일을 프로그램 배포자에게 전달하면 원인 파악에 도움이 돼요. (상담 내용은 이 파일에 기록되지 않아요.)

파일 위치:

C:\Users\사용자이름\gongulbaki_debug.txt

파일 찾는 방법:

  1. ①파일 탐색기를 열어주세요 (윈도우 키 + E)
  2. ②상단 주소창을 클릭하고 %USERPROFILE% 를 입력 후 Enter
  3. ③gongulbaki_debug.txt 파일을 찾아 배포자에게 전달해주세요
